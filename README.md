# EXCEL-SALES-ANALYTICS
Introduction
The Sales Analytics Project aims to provide a comprehensive analysis of sales data using Excel. The goal is to identify trendsto help stakeholders make data-driven decisions.

Data Analysis
The analysis is conducted using various Excel features and functions, including:

Data Cleaning:

Removing duplicates
Handling missing values
Correcting data types
Descriptive Statistics:

Calculating summary statistics (mean, median, mode, etc.)
Analyzing sales distribution

Sales Trends:

Identifying seasonal trends
Analyzing sales over time

Product Performance:

Identifying best-selling products
Analyzing product-wise sales

Customer Analysis:

Segmenting customers based on purchase behavior
Analyzing customer retention
